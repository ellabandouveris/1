Drive My Car

October 2022.
License: Laser cut 1/4 inch MDF with scans of my Driver's License pasted to both sides. With Arduino Adafruit with an accelerometer, USB extension cord, USB to USBc adapter.

Drive My Car is a unique video game controller. Using an accelerometer built into an Arduino Adafruit, I converted degree angles into button presses. If the user tilts the Driver's License >90ยบ to the left, the code essentially pushes the Left key. Game used: slow roads. Created in the UCLA Design Media Arts Fabrication Lab.
